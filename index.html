My Work

Crowd Counting using YOLOv5

  • We developed a crowd monitoring and management system using ESP32CAM and YOLOv5.
  • The system aims to count and prevent overcrowding to avoid stampedes.
  • For the initial phase, we used a scaled-down version where ball bearings represented people, training a custom YOLO model on a ball bearing dataset.
  • The setup includes an Arduino, a servo, and ESP32CAM. The camera scans the area continuously.
  • If the number of ball bearings exceeds a predefined threshold, indicating overcrowding, the Arduino triggers the servo to stop the gates.
  • Our project achieved remarkable success and secured the first position in a hackathon that involved more than 50 competing teams.
